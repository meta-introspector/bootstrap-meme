# Cargo.toml

**Type:** Configuration File

**Description:** The manifest file for the `emojitape_interpreter` Rust project. It defines project metadata, dependencies, features, and how the project should be built.

**Key Contents/Declarations:**
- `package`: Defines the project name, version, and edition.
- `dependencies`: Lists external Rust crates used by the project (e.g., `strum`, `strum_macros`).

**Relationships:**
- Directly influences the build process of all Rust source files in `src/`.
- Specifies the versions of libraries used by the Rust code.

**Keywords/Tags:** Rust, Cargo, configuration, dependencies, build system
