# onboarding.md - Conceptual Multivector Analysis

## Summary
This document provides an "Onboarding for New Users" guide, outlining initial steps to get started with the Gemini CLI and understand the project's workflow. It instructs users to install the Gemini CLI, introduces daily workflow acronyms (GM, WB, LG) for communication and task management, and emphasizes the importance of reviewing Standard Operating Procedures (SOPs) located in the `docs/sops/` directory.
