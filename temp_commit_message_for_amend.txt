feat: Implement Rust to Emoji Conversion Demo and Refactorings

This commit introduces a demonstration of Rust code to emoji conversion and back, along with significant refactorings to improve modularity and adhere to the "additive only" development principle (with a specific relaxation for Cargo.toml).

Key changes include:

- **ITIL Project Plan:** Documented the ITIL project plan for Rust to Emoji Language Conversion in `docs/plan/syn.md`, outlining objectives, requirements, and success criteria.
- **Modular Refactoring:**
    - Created `src/cocci_parser/`: A new module responsible for parsing `.cocci` file content into a structured `CocciElement` representation.
    - Created `src/rust_to_emoji/`: A new module designed to encapsulate Rust AST parsing and conversion to emoji tokens.
    - Refactored `src/cocci_converter.rs`: Now orchestrates the conversion process by utilizing `cocci_parser` and `rust_to_emoji`.
- **Rust to Emoji Conversion Demo:**
    - Implemented a standalone demonstration in `src/main_conversion_demo.rs` that reads sample Rust code, converts it to emoji tokens (using a temporary workaround for `syn::visit` issues), converts it back to Rust, and writes the regenerated code to `output_rewritten_rust.rs`.
    - Created `src/rust_to_emoji_workaround.rs` to house the current placeholder Rust-to-Emoji conversion logic, bypassing detailed `syn` AST traversal due to persistent compilation errors with `syn::visit`.
    - Created `src/emoji_to_rust_standalone.rs` for the reverse conversion.
    - Added `temp_test_conversion.rs` as a sample input file for the demo.
- **Cargo Configuration:**
    - Updated `Cargo.toml` to include `main_conversion_demo` as a new binary target, demonstrating a specific relaxation of the "additive only" constraint to enable project execution.
    - Ensured `syn` and `quote` dependencies are correctly configured.

This checkpoint establishes a working end-to-end flow for the Rust-to-Emoji conversion concept, laying the groundwork for future detailed semantic parsing and code generation.
